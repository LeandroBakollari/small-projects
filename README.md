# Small Projects

A collection of small browser projects. The root `index.html` page shows the four projects in a simple card grid with three projects per row on wider screens.

## Projects

### Calxy

Calxy is a local-only React app for planning study or work focus sessions. It lets users create sessions, view session history, and keep preferences in browser `localStorage` without a backend.

Run it locally from the `Calxy` folder:

```sh
npm install
npm run dev
```

### Pixel Painting

Pixel Painting is a small drawing tool where users paint pixel art on a square grid. It includes a size slider, color picker, paint mode, eraser mode, rainbow mode, grid-line toggle, and clear button.

Open it from `pixel-painting/index.html`.

### Rock, Paper, Scissors Battle

Rock, Paper, Scissors Battle is a two-player browser game. Each player locks in a move, the hands animate, the winner is scored, and players can use either on-screen buttons or keyboard controls.

Open it from `rock-paper-scissors-battle/index.html`.

### Reflex Circles

Reflex Circles is a reaction game where users click circles or press matching keys before the circles disappear. The game tracks score, combo, health, and gets faster as the round continues.

Open it from `wack-a-mole-game/index.html`.
