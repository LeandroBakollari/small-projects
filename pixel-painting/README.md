# Pixel Painting

Pixel Painting is a small browser drawing game where you paint on a square grid one pixel at a time.

## How it works

Choose a color, then click and drag across the board to paint pixels. The size slider changes the board from a small 8 x 8 grid up to a detailed 64 x 64 grid. Changing the size creates a fresh blank board.

Use Paint mode to draw with the selected color. Use Eraser mode to remove color from pixels, or right-click while in Paint mode to erase temporarily. Rainbow mode paints each pixel with a random bright color. Grid Lines can be turned on or off to make the pixel cells easier to see.

The Clear button wipes the board back to a blank canvas.

## Keyboard shortcuts

- `P`: Paint mode
- `E`: Eraser mode
- `R`: Rainbow mode
- `G`: Show or hide grid lines
- `C`: Clear the board
